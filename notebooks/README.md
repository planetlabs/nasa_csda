# Planet Satellite Data Quickstart

Notebooks to quickly get you started with querying and downloading Planet data through the CSDA program.

These notebooks are designed to be both tutorials in acquiring Planet Satellite data, as well as templates for downloading data at a location of your choice. For more in depth training on Planet's various API's, please visit the [public notebooks repo](https://github.com/planetlabs/notebooks).
